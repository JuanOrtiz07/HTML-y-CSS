Imágenes en HTML

Las imagenes son unas etiquetas que se auto-cierran y se utilizan para insertar imágenes en una página web.

El atributo src es el que se encarga de colocar la ruta de la imagen, el atributo alt es el que se encarga de colocar un texto alternativo a la imagen, y el atributo title es el que se encarga de colocar un texto que se muestra cuando pasamos el cursor sobre la imagen

Ahora, para mostrar una imagen que no esta en la misma carpeta del archivo HTML, debemos especificar la ruta relativa o absoluta hacia esa imagen.
