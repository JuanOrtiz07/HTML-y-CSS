Selectores CSS

Los selectores CSS son patrones utilizados para seleccionar y manipular elementos HTML en una página web. Permiten aplicar estilos específicos a elementos individuales o grupos de elementos según sus atributos, clases, identificadores, relaciones jerárquicas y más. Algunos ejemplos de selectores CSS incluyen:

  • Selector de tipo: Selecciona elementos por su nombre de etiqueta, como p para párrafos o h1 para encabezados.
  • Selector de clase: Selecciona elementos que tienen una clase específica, utilizando un punto seguido del nombre de la clase, como .mi-clase.
  • Selector de ID: Selecciona un elemento único con un identificador específico, utilizando un signo de número seguido del ID, como #mi-id.
  • Selector de atributo: Selecciona elementos basados en la presencia o valor de un atributo, como [type="text"].
  • Selector descendiente: Selecciona elementos que son descendientes de otro elemento, como div p, que selecciona todos los párrafos dentro de un div.
  • Selector hijo: Selecciona elementos que son hijos directos de otro elemento, como ul > li, que selecciona solo los elementos li que son hijos directos de ul.

    • Selectores avanzados

    Además de los selectores básicos, CSS ofrece selectores avanzados que permiten una mayor precisión en la selección de elementos. Algunos ejemplos incluyen:

    • Selector de pseudo-clase: Selecciona elementos en un estado específico, como :hover para aplicar estilos cuando el usuario pasa el cursor sobre un elemento.
    • Selector de pseudo-elemento: Permite seleccionar y estilizar partes específicas de un elemento, como ::before y ::after.
    • Selector de grupo: Permite aplicar estilos a múltiples elementos a la vez, separando los selectores con comas, como h1, h2, h3.
    • Selector de hermano adyacente: Selecciona un elemento que es inmediatamente precedido por otro elemento, como h1 + p.
    • Selector de hermano general: Selecciona todos los elementos que son hermanos de otro elemento, como h1 ~ p.
