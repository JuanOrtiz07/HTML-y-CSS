Enlaces que se abren en una nueva pestaña

GitHub pero en otra pestaña

para abrir un enlace en una nueva pestaña, utilizamos el atributo target="_blank"

Atributos de los elementos

para agregar un atributo Title a un elemento debemos usar el atributo title="valor", ejemplo, al enlace de GitHub: le podemos poner despues del href un title="Repositorio en GitHub de HTML"

esto se muestra cuando pasamos el cursor sobre el enlace, y se utiliza para proporcionar información adicional sobre el elemento al cual coloquemos el atributo
