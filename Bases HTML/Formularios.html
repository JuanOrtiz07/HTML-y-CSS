El elemento form se utiliza para crear formularios en HTML, dentro de este elemento podemos colocar diferentes tipos de campos de entrada, como por ejemplo, campos de texto, campos de correo electrónico, campos de contraseña, botones de envío, casillas de verificación, botones de opción y campos para subir archivos. Cada uno de estos campos tiene un atributo type que especifica el tipo de campo que se va a crear. Además, algunos campos pueden tener atributos adicionales como required para indicar que el campo es obligatorio o placeholder para mostrar un texto de ejemplo dentro del campo.
