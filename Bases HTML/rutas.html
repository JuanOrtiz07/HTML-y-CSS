Rutas en HTML

Las rutas en HTML se utilizan para especificar la ubicación de los archivos y recursos en una página web.

En este ejemplo, la imagen se encuentra dentro de una carpeta llamada "imagenes", y la ruta relativa hacia esa imagen es "imagenes/image.png".

Para acceder a un archivo en una carpeta diferente, se debe colocar "nombre_carpeta/archivo_deseado". si por otro lado esta fuera d ela carpeta que estamos debemos colocar "../archivo_deseado" de esta manera retrocedemos
